# odin-admin-dashboard

A full scale dashboard user interface built with **CSS Grid**. This project focuses on complex layouts, alignment, and maintaining a clean component structure without using frameworks.

# Live Demo:
https://alec1999.github.io/odin-admin-dashboard/

# Features:
• **Grid-first layout**: Utilized CSS Grid for layout that adapts to smaller screens.
• **SVG system**: Implemented icon system for each section of dashboard.
• **Box-shadow styling**: Project cards with custom shadow depth.

# What I Learned:
• How to manage `z-index` and `clip-path` to create clean UI depth.
• Using CSS Grid to create a multi-layered grid.
• How to implement SVGs and icons into a project.
